## Controls

### 🚁 Helicopter Flight

| Key | Action |
| --- | ------ |
| **W** | Accelerate and move forward |
| **S** | Decelerate and move backward |
| **A** | Rotate left |
| **D** | Rotate right |

### 🛫 Take-off & Landing

| Key | Action |
| --- | ------ |
| **P** | Lift off from building or lake (deploys bucket on take-off from building) |
| **L** | Land on building (retracts bucket) or on lake (fills bucket with water) |

### 🔥 Fire Fighting

| Key | Action |
| --- | ------ |
| **O** | Open bucket valve and drop water on the fire (only when bucket is filled) |

### ⚙️ General

| Key | Action |
| --- | ------ |
| **R** | Reset the scene (returns helicopter to starting position) |

### 🎛️ GUI Controls

| Control | Description |
| ------- | ----------- |
| **Display Axis** | Toggle world axis visibility |
| **Display Earth** | Toggle earth sphere visibility |
| **Display Plane** | Toggle ground plane visibility |
| **Display Panorama** | Toggle panorama background visibility |
| **Display Building** | Toggle firefighters building visibility |
| **Display Forest** | Toggle forest visibility |
| **Display Helicopter** | Toggle helicopter visibility |
| **Display Lake** | Toggle lake visibility |
| **Display Fire** | Toggle fire and fireplace visibility |
| **Speed Factor** | Adjust helicopter rotation speed (0.1 – 3.0) |
